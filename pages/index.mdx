## Quick Start

### Installation

The Framer API was designed to work closely with [Framer](https://framer.com), but is great as a standalone library. You can install it via [npm](https://www.npmjs.com/), via your Command Line.

```jsx
// Install Framer via command line
npm install framer
```

### Importing

Once installed, you can import Framer into your files.

```jsx
// Import from Framer
import { Frame, Scroll, useCycle } from "framer"
```

## Basics

### Animate

<div>

Letâ€™s have a look at a simple animation that scales a `Frame` to half the size. Notice how you can mix `Frame` elements with regular HTML, just like you would do within any React component.

<CTA name="Learn More" link="/legacy/docs/examples/#animate" />

</div>

```jsx
export function MyComponent() {
  return <Frame animate={{ scale: 0.5 }} />
}
```

---

### Cycle

A more extensive example would be to toggle the `scale` value on a tap, which uses the `useCycle` hook to cycle through a set of values.

```jsx
// Cycle scale on tap
export function MyComponent() {
  const [scale, cycle] = useCycle(3, 1)
  return (
    <Frame
      animate={{ scale: scale }}
      onTap={() => cycle()}
    />
  )
}
```

<div>

You can also use `useCycle` to cycle between sets of visual properties.
Here, weâ€™re also rotating the `Frame` on every tap, next to scaling it.

<CTA name="Learn More" link="/legacy/docs/examples/#cycle" />

</div>

```jsx
// Cycle multiple properties on tap
export function MyComponent() {
  const [twist, cycle] = useCycle(
    { scale: 0.5, rotate: 0 },
    { scale: 1, rotate: 90 }
  )

  return <Frame animate={twist} onTap={() => cycle()} />
}
```

### Dragging

<div>

You can easily make any `Frame` draggable by using the `drag` property.
To constrain dragging to a vertical or horizontal axis, you can include an optional `"x"` or `"y"` value.

<CTA name="Learn More" link="/legacy/docs/examples/#dragging" />

</div>

```jsx
// Drag Frame in any direction
export function MyComponent() {
  return <Frame drag />
}
```

---

### Scrolling

<div>

For scrolling, you can wrap any content inside a `Scroll` component, which exceeds the size of the component either vertically or horizontally.

<CTA name="Learn More" link="/legacy/docs/examples/#scrolling" />

</div>

```jsx
// Scroll two Frames
export function MyComponent() {
  return (
    <Scroll>
      <Frame />
      <Frame />
    </Scroll>
  )
}
```
